Dado los resultados obtenidos en la primera parte del parcial se puede notar que la curva de crecimiento de los tiempos de los diferentes algoritmos es diferente para cada uno.
Esto dado que se realizan mas bucles en algunos algoritmos que en otros.

En el caso de las busquedas el algoritmo que mejor se desempeno fue el Sequencial teniendo este una complejidad de O(n).
Ya que no requiere bucles y solo realiza el recorrido del arreglo para hallar el dato buscado.

Por otro lado, en el caso de los algoritmos de ordenamiento, el que mejor se desmepeno fue el de Seleccion.
esto porque no vuelve a hacer chequeos de los elementos ya organizados, haciendo que los futuros bucles de organizacion sean cada vez mas cortos,
este algoritmo posee una complejidad de O(n^2).

Por este motivo, los algoritmos de Seleccion y de Secuencia fueron usados con el archivo excel en la parte 2, pero con modificaciones 
para poder realizar la lectura correcta dentro de las columnas correspondientes.
